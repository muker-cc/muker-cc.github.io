推荐一个 Java 企业信息化系统

397 阅读 0 评论 0 点赞

推荐一个 OA 开源项目：O2OA，该应用是基于 JavaEE 技术栈搭建的适用于企业OA、协同办公类信息化的平台。

采用分布式架构设计，是一款真正全代码开源的企业应用定制化开发平台。

这个开源项目包含的核心功能有：

  • 流程管理
  • 门户管理
  • 信息管理
  • 数据管理

  • 服务管理

除此之外，平台还提供了完整的用户管理，权限管理，流程和信息管理体系，并且提供了大量的开发组件和开箱即用的应用，可以大幅度减化企业信息化建设成本和业务应用开发难度。

除了基础的企业信息化服务，还有语音办公、人脸识别、指纹认证、智能文档纠错、智能填表推荐等特色功能，并且移动端已经与企业微信、钉钉集成。
配备多种企业信息化服务
自动适配移动端
与钉钉、企业微信集成
提供丰富的应用模块
搭建门户网站

这个开源项目可以关注公众号「逛逛GitHub」回复「0703」下载。
