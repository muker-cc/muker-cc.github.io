这 2 个工作流自动化工具 yyds

138 阅读 0 评论 0 点赞

很多读者可能听说过工作流，但是不知道这是什么东西。简单来说，它是一个与其他应用集成的应用程序，目标是自动化各应用之间的流程。
有一款非常火的工作流自动化工具在 GitHub 上开源了，这个名为 n8n 工作流项目在 GitHub 上斩获了 17K 的 Star。
比如你需要从数据库中获取一些数据后下载为 excel 然后通过邮件发送给其他人，这个场景就是一个流程，完全可以通过 n8n 实现流程的自动化，它作为一个桥梁，将数据库 Excel 邮箱这三个应用程序连接起来。
你想将什么应用程序串联起来，可以通过开源工具的平台自定义配置。
不仅仅是数据库、邮箱，开源工具 n8n 支持 200 多种不同的应用程序，如果你明确知道自己想要的功能也可以集成几乎任何具有 API 的应用程序，从而充分利用 n8n 的开源和多功能特性。
n8n 的工作原理也非常简单，上图中这些应用都会开放一些 API ，通过应用程序的 API 连接到应用程序就是 n8n 开源工具的职责。

国内的工作流工具
n8n 是国外的工作流自动化工具，集成的应用程序都是国外的产品，比如 Facebook、Airtable、Google doc 等等。显然，这款开源工具在国内没有使用场景，毕竟大部分国内的用户都是用本土的互联网产品。
疫情过后，国内企业有了线上办公的趋势，线上办公就会用到很多产品，比如文档、会议、CRM、销售系统等等产品，但是各产品之间相互割裂，数据流转需要人工介入，照成效率低下。
国内大厂也在布局流程自动化，为企业做数字化转型，比如腾讯出品的腾讯千帆场景连接器可以将国内的应用互联互通，包括企业微信、腾讯问卷、腾讯文档、维格表、金数据等。
腾讯千帆场景连接器提供了一些常用的场景模板，只需要通过傻瓜式的拖拉拽操作就能快速配置下面的功能。
1. 当用户在【腾讯问卷】填写活动报名信息后，自动将信息写入【维格表】中进行整合归档，同时通过【企业微信】通知对应人员及时跟进。
2. 当客户在【腾讯问卷】填写活动报名信息后，实时通过【企业微信群机器人】通知对应销售人员及时跟进。
3. 当有客户提交【腾讯问卷】时，自动将客户信息写入【维格表】中以便营销人员整合归档。
4. 当企业员工更新【腾讯文档】内容时，通过【企业微信自建应用】推送员工操作记录。
...
还有其他玩法，这个产品已经开始进行内测，内测期间名额有限，如果有想要申请试用的读者可以电脑网页端登录：https://ssp.cloud.tencent.com 进行申请，同时可加入腾讯千帆场景连接器的企业微信群，进行沟通交流。
