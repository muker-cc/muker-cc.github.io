这个 GitHub 项目让你任务栏变透明

461 阅读 0 评论 0 点赞

GitHub 上除了实战项目、顶级开源项目、学习资源外，还隐藏着一些实用工具，老逛就来扒一扒 GitHub 上那些高赞的轻量级工具。
01.  任务栏变透明
Windows 10 系统自带的任务栏设置，仅仅支持更改主题色(且必须跟随系统)，自动隐藏，调整位置等基础功能：
动图：系统自带的任务栏设置
而独立调整它的颜色、强度和不透明度等,都没办法实现。我的任务栏是透明的，结合动态壁纸就会营造出一种沉浸感。
‍‍‍‍‍‍这种效果可以借助一个 GitHub 开源项目实现，TranslucentTB 可以将你的任务栏透明化。安装后，你就能控制 Windows 任务栏的外观。
软件非常轻量，运行时系统占用资源极小，内存方面，仅占用不足 4MB ，耗电则可以忽略不计。有多种自定义选项，可以让你去调整诸如颜色和效果之类的表现。
比如有以下几种优化模式：
  • 轻微毛玻璃效果
  • 全透明效果
  • Windows原生效果
  • 完全不透明效果  

  • 遵循Fluent Design的效果

02.  美化效果预览
毛玻璃效果
大家都知道，Win10 原生的状态栏实际也是有些许的毛玻璃效果，而工具中的毛玻璃效果更加激进一些，更通透：
动图：开启毛玻璃效果之后
Fluent Design
微软 Build 2017 第二天的主题演讲中，微软拿出了全新的设计范式，为的就是彻底改变沿用多年的平面矩形UI。
微软将这套全新的设计语言名为 Fluent Design System（流畅设计体系），该体系共有五大核心元素，包括Light（光感）、Depth（深度）、Motion（动画）、Material（材质）和 Scale（缩放）
动图：开启 Fluent Design 效果后
全透明效果
任务栏彻底透明，这也是很多同学想要的效果。
 动图：开启全透明效果后
