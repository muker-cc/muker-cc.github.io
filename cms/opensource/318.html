人生重开模拟器 牛啊

259 阅读 0 评论 0 点赞

前段时间，有一个叫做人生重开模拟器的开源游戏刷屏全网。短短十天在 GitHub 斩获  8.3 K 的 Star。

三天就创造了 2 亿的访问量。


玩法相当简单，刚开始 只需要你抽几张「天赋」卡，为自己的「颜值、智力、体质、家境」设定初始值，你的人生路就开始了。

一分钟，让你提前看遍人生无常。


先抽天赋卡，我选择活死人、驻颜、智可生财


调整自己的颜值、智力、体质、家境属性值，你的人生开始了！
