微软正式推出第一台云电脑 Windows 365

547 阅读 0 评论 0 点赞


现在你可以把手机变成 Windows 电脑，随时随地工作了。

”


作者 | 冉启行

编辑 | 李帅飞

雷锋网消息，7 月 14 日晚间，微软正式发布了 Windows 365——云电脑（Cloud PC）。

据官方介绍，这一云服务将 Windows 操作系统带到了微软智能云端上，用户可以通过串流的方式在个人或公司设备上访问并使用该到系统。

微软公司董事长兼首席执行官萨提亚 · 纳德拉（Satya Nadella）对此表示：

我们以 Windows 365 创造出了一个全新的类别：Cloud PC。如同以SaaS方式将应用程序带到云端一样，我们现在把操作系统也带到了云上，这将为组织提供一种更加灵活、安全的工作方式，让员工无论置身何地，都能保持连接，并更好地释放生产力。

按照微软官方说法，Windows 365 具体应用功能和特性如下：

  • 完整的 PC 体验：Windows 365 将提供云端的完整 PC 体验，支持 Windows 10 及 Windows 11（在2021年晚些时候正式商用后生效）。

  • 数据传输：连接到 Cloud PC 后，用户可以从云端将应用、工具、数据和设置传输到任意设备。

  • 合理调配资源：由于 Cloud PC 的云端特性，用户的云端存储空间是可以根据需求自由调配大小的，公司可以按需付费。

  • 安全保护：Windows 365从设计源头确保了安全性，用户信息受到保护，并被存储在云端，而非设备端。

  • 跨设备登录：Cloud PC 允许用户在不同设备上登录，仍能获取相同的内容，帮助用户提高工作效率。

  • 企业为主：Cloud PC 的商用客户将主要以企业为主。

这意味着即使员工在家办公也能随时访问到存储在公司电脑上的数据，灵活性工作，由于数据存储在云端，还能大大增强其数据的安全性和保密性，公司在资源分配上面也有了更加清晰的逻辑。

某种程度上，云电脑的出现，也是在顺应企业进行数字化转型的大趋势，并且它也可以摆脱对繁重的主机设备的依赖。

对于微软 Windows 365，埃维诺全球现代工作场所负责人 Veit Siegenheim 表示，“借助 Windows 365，微软正在利用智能云在任何地方、任何设备上提高生产力，并提升工作场所体验。”

不过，虽然 Windows 365 的跨设备登录是该服务一大亮点之一，但是，一位正在使用云电脑的项目工程师告诉雷锋网：虽然云电脑允许用户在手机、平板等移动设备上访问到公司电脑，但是就操作性而言，还是 PC 更占优势。

雷锋网了解到，由于云电脑存储的数据都在云服务器上，所以对于用户网络的要求也会比较高，在一些信号较差的地方工作时，将会有非常大的阻碍。

Cloud PC 是一种全新的混合个人运算服务，它能够带来的高效、便捷的工作体验这一点毋庸置疑，但就其普及性来说，还需要时间来考察。

最后，微软将在 2021 年 8 月 2 日起，正式将Windows 365 商用到各种规模的企业中去，不过目前在定价方面暂未进一步透露。
